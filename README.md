# Project Nova

A custom Bluetooth game controller — drift-free TMR joysticks, 6-axis gyro, multi-device virtual partitioning, and a dual XInput/DS4 boot toggle. Built on an Xbox-style shell targeting a 15–20 hour battery life.

- [Master Specification](SPEC.md) — full engineering spec, firmware architecture, and decisions log
